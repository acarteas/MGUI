# Image

Displays texture content with configurable source, tint, and stretch mode.

- XAML tag: `<Image>`
- Runtime type: `MGUI.Core.UI.MGImage`
- Base type: [Element](./element.md)
- Element type: `MGElementType.Image`
- Content model: No child elements.
- Inherits shared API from [Element](./element.md).

## Direct Properties

| Property | Type |
| --- | --- |
| `SourceName` | `string` |
| `Source` | `MGTextureData?` |
| `TextureColor` | `XAMLColor?` |
| `Stretch` | `Stretch?` |
