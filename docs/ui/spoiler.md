# Spoiler

Reveal/hide control that swaps placeholder text for protected content.

- XAML tag: `<Spoiler>`
- Runtime type: `MGUI.Core.UI.MGSpoiler`
- Base type: [SingleContentHost](./single-content-host.md)
- Element type: `MGElementType.Spoiler`
- Content model: Single `Content` child that is revealed when the spoiler is open.
- Inherits shared API from [SingleContentHost](./single-content-host.md).

## Direct Properties

| Property | Type |
| --- | --- |
| `Button` | `Button` |
| `UnspoiledText` | `string` |
| `UnspoiledTextAlignment` | `HorizontalAlignment?` |
| `IsRevealed` | `bool?` |
