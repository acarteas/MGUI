# ContentPresenter

Lightweight single-content wrapper used when you need a stand-alone content host.

- XAML tag: `<ContentPresenter>`
- Runtime type: `MGUI.Core.UI.MGContentPresenter`
- Base type: [SingleContentHost](./single-content-host.md)
- Element type: `MGElementType.ContentPresenter`
- Content model: Single `Content` child.
- Inherits shared API from [SingleContentHost](./single-content-host.md).

## Direct Properties

This type does not add direct properties beyond those inherited from [SingleContentHost](./single-content-host.md).
