# MultiContentHost

Abstract element base for controls that host a child collection.

- XAML tag: `abstract`
- Runtime type: `MGUI.Core.UI.Containers.MGMultiContentHost`
- Base type: [Element](./element.md)
- Content model: A collection of child elements assigned through `Children` or a specialized content property such as `Items` or `Tabs`.
- Inherits shared API from [Element](./element.md).

## Direct Properties
