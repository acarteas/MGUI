# StackPanel

Layout container that stacks children horizontally or vertically.

- XAML tag: `<StackPanel>`
- Runtime type: `MGUI.Core.UI.MGStackPanel`
- Base type: [MultiContentHost](./multi-content-host.md)
- Element type: `MGElementType.StackPanel`
- Content model: Collection of `Children` stacked along `Orientation`.
- Inherits shared API from [MultiContentHost](./multi-content-host.md).

## Direct Properties

| --- | --- |
| `Border` | `Border` |
| `Orientation` | `Orientation?` |
| `Spacing` | `int?` |
