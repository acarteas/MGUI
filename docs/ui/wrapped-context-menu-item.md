# WrappedContextMenuItem

Abstract context-menu item base that can optionally own a nested submenu.

- XAML tag: `abstract`
- Runtime type: `MGUI.Core.UI.MGWrappedContextMenuItem`
- Base type: [ContextMenuItem](./context-menu-item.md)
- Element type: `MGElementType.ContextMenuItem`
- Content model: Uses inherited `Content` for the visible body and adds optional nested submenu support.
- Inherits shared API from [ContextMenuItem](./context-menu-item.md).
