# SingleContentHost

Abstract element base for controls that expose exactly one primary child element.

- XAML tag: `abstract`
- Runtime type: `MGUI.Core.UI.Containers.MGSingleContentHost`
- Base type: [Element](./element.md)
- Content model: A single child assigned through the `Content` property or the XAML content slot.
- Inherits shared API from [Element](./element.md).
