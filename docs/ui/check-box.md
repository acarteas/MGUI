# CheckBox

Toggle control with checkbox glyph, optional three-state support, and inline content.

- XAML tag: `<CheckBox>`
- Runtime type: `MGUI.Core.UI.MGCheckBox`
- Base type: [SingleContentHost](./single-content-host.md)
- Element type: `MGElementType.CheckBox`
- Content model: Single `Content` child shown beside the checkbox glyph.
- Inherits shared API from [SingleContentHost](./single-content-host.md).
