# OverlayPanel

Layout container that layers multiple children in the same arrangement space.

- XAML tag: `<OverlayPanel>`
- Runtime type: `MGUI.Core.UI.MGOverlayPanel`
- Base type: [MultiContentHost](./multi-content-host.md)
- Element type: `MGElementType.OverlayPanel`
- Content model: Collection of layered `Children` optionally offset with attached properties.
- Inherits shared API from [MultiContentHost](./multi-content-host.md).

## Direct Properties

This type does not add direct properties beyond those inherited from [MultiContentHost](./multi-content-host.md).
