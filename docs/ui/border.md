# Border

Wraps one child element with border styling and optional background behavior.

- XAML tag: `<Border>`
- Runtime type: `MGUI.Core.UI.MGBorder`
- Base type: [SingleContentHost](./single-content-host.md)
- Element type: `MGElementType.Border`
- Content model: Single `Content` child.
- Inherits shared API from [SingleContentHost](./single-content-host.md).

## Direct Properties

| Property | Type |
| --- | --- |
| `BorderBrush` | `BorderBrush` |
| `BorderThickness` | `Thickness?` |
