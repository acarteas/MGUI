# GridSplitter

Resize handle that adjusts adjacent rows or columns in a `Grid`.

- XAML tag: `<GridSplitter>`
- Runtime type: `MGUI.Core.UI.MGGridSplitter`
- Base type: [Element](./element.md)
- Element type: `MGElementType.GridSplitter`
- Content model: No child elements.
- Inherits shared API from [Element](./element.md).
