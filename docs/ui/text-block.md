# TextBlock

Displays formatted text, including optional inline formatting and progressive reveal.

- XAML tag: `<TextBlock>`
- Runtime type: `MGUI.Core.UI.MGTextBlock`
- Base type: [Element](./element.md)
- Element type: `MGElementType.TextBlock`
- Content model: Content property is `Text`.
- Inherits shared API from [Element](./element.md).
