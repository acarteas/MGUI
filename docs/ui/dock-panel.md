# DockPanel

Layout container that docks children to edges and can let the last child fill.

- XAML tag: `<DockPanel>`
- Runtime type: `MGUI.Core.UI.MGDockPanel`
- Base type: [MultiContentHost](./multi-content-host.md)
- Element type: `MGElementType.DockPanel`
- Content model: Collection of `Children` positioned by attached `Dock` values.
- Inherits shared API from [MultiContentHost](./multi-content-host.md).
