# Slider

Draggable numeric selector with optional ticks, thumb styling, and discrete steps.

- XAML tag: `<Slider>`
- Runtime type: `MGUI.Core.UI.MGSlider`
- Base type: [Element](./element.md)
- Element type: `MGElementType.Slider`
- Content model: No child elements.
- Inherits shared API from [Element](./element.md).
