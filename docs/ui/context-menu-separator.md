# ContextMenuSeparator

Separator entry used to divide groups inside a `ContextMenu`.

- XAML tag: `<ContextMenuSeparator>`
- Runtime type: `MGUI.Core.UI.MGContextMenuSeparator`
- Base type: [ContextMenuItem](./context-menu-item.md)
- Element type: `MGElementType.ContextMenuItem`
- Content model: No normal child content; visual separator is configured through the `Separator` property.
- Inherits shared API from [ContextMenuItem](./context-menu-item.md).

## Direct Properties

| Property | Type |
| --- | --- |
| `Separator` | `Separator` |
