# RadioButton

Mutually exclusive option control that participates in a named radio-button group.

- XAML tag: `<RadioButton>`
- Runtime type: `MGUI.Core.UI.MGRadioButton`
- Base type: [SingleContentHost](./single-content-host.md)
- Element type: `MGElementType.RadioButton`
- Content model: Single `Content` child shown beside the radio glyph.
- Inherits shared API from [SingleContentHost](./single-content-host.md).
