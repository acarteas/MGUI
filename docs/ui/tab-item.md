# TabItem

One tab entry inside a `TabControl`, containing a header and content body.

- XAML tag: `<TabItem>`
- Runtime type: `MGUI.Core.UI.MGTabItem`
- Base type: [SingleContentHost](./single-content-host.md)
- Element type: `MGElementType.TabItem`
- Content model: Single `Content` child plus a separate `Header` element.
- Inherits shared API from [SingleContentHost](./single-content-host.md).

| Property | Type |
| --- | --- |

| `IsTabSelected` | `bool?` |
