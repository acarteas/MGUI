# TreeViewItem

Expandable node inside a `TreeView` that can hold child items.

- XAML tag: `<TreeViewItem>`
- Runtime type: `MGUI.Core.UI.MGTreeViewItem`
- Base type: [MultiContentHost](./multi-content-host.md)
- Element type: `MGElementType.TreeViewItem`
- Content model: Content property is `Children`; `Header` is a separate property.
- Inherits shared API from [MultiContentHost](./multi-content-host.md).

## Direct Properties

| Property | Type |
| --- | --- |
| `Header` | `string` |
| `IsExpanded` | `bool?` |
