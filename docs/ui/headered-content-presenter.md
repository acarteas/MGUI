# HeaderedContentPresenter

Presents a header element next to or above a single content element.

- XAML tag: `<HeaderedContentPresenter>`
- Runtime type: `MGUI.Core.UI.MGHeaderedContentPresenter`
- Base type: [SingleContentHost](./single-content-host.md)
- Element type: `MGElementType.HeaderedContentPresenter`
- Content model: Single `Content` child plus a separate `Header` element.
- Inherits shared API from [SingleContentHost](./single-content-host.md).

## Direct Properties

| Property | Type |
| --- | --- |
| `Header` | `Element` |
| `HeaderPosition` | `Dock?` |
| `Spacing` | `int?` |
