# ContextualContentPresenter

Switches between true and false content branches from a boolean value.

- XAML tag: `<ContextualContentPresenter>`
- Runtime type: `MGUI.Core.UI.MGContextualContentPresenter`
- Base type: [Element](./element.md)
- Element type: `MGElementType.ContextualContentPresenter`
- Content model: Two optional branches: `TrueContent` and `FalseContent`.
- Inherits shared API from [Element](./element.md).

## Direct Properties

| Property | Type |
| --- | --- |
| `Value` | `bool` |
| `TrueContent` | `Element` |
| `FalseContent` | `Element` |
