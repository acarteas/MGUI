# ContextMenuItem

Abstract base for entries that live inside a `ContextMenu`.

- XAML tag: `abstract`
- Runtime type: `MGUI.Core.UI.MGContextMenuItem`
- Base type: [SingleContentHost](./single-content-host.md)
- Element type: `MGElementType.ContextMenuItem`
- Content model: A single `Content` child used as the visible body of the menu entry.
- Inherits shared API from [SingleContentHost](./single-content-host.md).

## Notes

- Concrete menu items add behavior such as commands, toggles, separators, and radio groups.

## Direct Properties

This type does not add direct properties beyond those inherited from [SingleContentHost](./single-content-host.md).
