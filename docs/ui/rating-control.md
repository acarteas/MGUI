# RatingControl

Interactive rating selector built from repeated shapes such as stars or hearts.

- XAML tag: `<RatingControl>`
- Runtime type: `MGUI.Core.UI.MGRatingControl`
- Base type: [Element](./element.md)
- Element type: `MGElementType.RatingControl`
- Content model: No child elements.
- Inherits shared API from [Element](./element.md).
