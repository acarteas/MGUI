# Separator

Draws a horizontal or vertical divider with configurable thickness.

- XAML tag: `<Separator>`
- Runtime type: `MGUI.Core.UI.MGSeparator`
- Base type: [Element](./element.md)
- Element type: `MGElementType.Separator`
- Content model: No child elements.
- Inherits shared API from [Element](./element.md).
