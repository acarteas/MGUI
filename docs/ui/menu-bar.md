# MenuBar

Top-level horizontal menu strip composed of `MenuBarItem` entries.

- XAML tag: `<MenuBar>`
- Runtime type: `MGUI.Core.UI.MGMenuBar`
- Base type: [SingleContentHost](./single-content-host.md)
- Element type: `MGElementType.MenuBar`
- Content model: Primary content property is `Items`.
- Inherits shared API from [SingleContentHost](./single-content-host.md).

## Direct Properties

| Property | Type |
| --- | --- |
| `Items` | `List<MenuBarItem>` |
| `ButtonWrapperTemplate` | `Button` |
| `ItemsPanel` | `StackPanel` |
