# ToggleButton

Button that persists a checked state and can restyle itself when selected.

- XAML tag: `<ToggleButton>`
- Runtime type: `MGUI.Core.UI.MGToggleButton`
- Base type: [SingleContentHost](./single-content-host.md)
- Element type: `MGElementType.ToggleButton`
- Content model: Single `Content` child.
- Inherits shared API from [SingleContentHost](./single-content-host.md).

## Direct Properties

| Property | Type |
| --- | --- |
| `Border` | `Border` |
| `CheckedBackgroundBrush` | `FillBrush` |
| `CheckedTextForeground` | `XAMLColor?` |
| `IsChecked` | `bool?` |
