# Rectangle

Simple filled and stroked rectangular primitive element.

- XAML tag: `<Rectangle>`
- Runtime type: `MGUI.Core.UI.MGRectangle`
- Base type: [Element](./element.md)
- Element type: `MGElementType.Rectangle`
- Content model: No child elements.
- Inherits shared API from [Element](./element.md).

## Direct Properties

| `StrokeThickness` | `int?` |
| `Fill` | `FillBrush` |
