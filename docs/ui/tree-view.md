# TreeView

Hierarchical container for expandable tree items with selection styling.

- XAML tag: `<TreeView>`
- Runtime type: `MGUI.Core.UI.MGTreeView`
- Base type: [MultiContentHost](./multi-content-host.md)
- Element type: `MGElementType.TreeView`
- Content model: Content property is `Children`.
- Inherits shared API from [MultiContentHost](./multi-content-host.md).

## Direct Properties

| Property | Type |
| --- | --- |
| `Border` | `Border` |
| `IndentSize` | `int?` |
| `SelectionBackgroundBrush` | `FillBrush` |
| `SelectionForeground` | `XAMLColor?` |
